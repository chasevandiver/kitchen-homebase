📅 Calendars
iCloud: Calendar app → Calendars → ⓘ → Share Calendar → Public Calendar ON → Share Link → replace webcal:// with https://

Google Calendar: calendar.google.com → Settings → your calendar → Integrate calendar → copy Secret address in iCal format

Add one entry per calendar — you can add multiple for the same person.
🖼️ iCloud Shared Album
iPhone: Photos app → Albums → your shared album → tap the person icon (top right) → toggle Public Website ON → Share Link → copy that URL and paste below.
Photos refresh every 6 hours automatically.
🎵 Spotify Now Playing
Shows the currently playing song as a mini-player on the dashboard.

One-time setup:
1. Go to developer.spotify.com → Log in → Create App
2. Add your Vercel URL + /api/spotify?action=callback as a Redirect URI
3. Copy your Client ID and Client Secret
4. Add SPOTIFY_CLIENT_ID, SPOTIFY_CLIENT_SECRET, FIREBASE_DB_URL, and FIREBASE_DB_SECRET to Vercel Environment Variables
5. Deploy api/spotify.js, then tap Connect below
